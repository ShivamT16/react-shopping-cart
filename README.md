## React Shopping Cart

### How to install and run locally ?

```
$ git clone https://github.com/ShivamT16/react-shopping-cart.git
$ cd react-shopping-app
$ npm install
$ npm start
```
### Features
Product Listing Page  
Product Detail Page  
Product Filters  
Search Functionality  
Cart Management

### Tech
React
